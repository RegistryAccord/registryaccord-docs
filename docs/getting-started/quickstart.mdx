This tutorial takes you from cloning to running the docs site and executing a minimal end‑to‑end flow.

## Prerequisites
- Node.js and a package manager.

## Steps
1. Clone the repo and install dependencies.
2. Start the docs site at http://localhost:3000.
3. Run the dev stack and verify health checks.
4. Create your first identity and publish a sample content record.
