Install the CLI/SDKs and set environment variables.

## CLI
- Install via npm (TypeScript) or download binaries if available.
- Verify version and help output.

## SDKs
- TypeScript: install package and initialize client.
- Go/Python: install module/package and configure base URL and auth.

## Config
- Environment variables for endpoints and credentials.
- Local vs remote environment notes.
