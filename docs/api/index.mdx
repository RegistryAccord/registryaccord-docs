# RegistryAccord API Reference

Complete API documentation for all RegistryAccord services.

## Available Services

### Identity Service
Authentication, authorization, and consent management.

**Base URL:** `https://api.registryaccord.com/identity/v1`

**Key Features:**
- WebAuthn/Passkey authentication
- OAuth2/OIDC flows
- Granular consent management
- Organization & team management

---

### Feed Generator Service
Ranking algorithms, fairness scorecards, and A/B testing.

**Base URL:** `https://api.registryaccord.com/feeds/v1`

**Key Features:**
- Pluggable ranking signals
- Fairness audit endpoints
- Dispute resolution system
- Public transparency scorecards

---

## Authentication

All API requests require authentication via JWT tokens.

Default rate limits apply to all endpoints. See individual service documentation for specific limits.
